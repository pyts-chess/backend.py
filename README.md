This project uses Python for the core logic and backend, and TypeScript for the frontend, offering a seamless and interactive chess experience.

## Features (In the works)
- **Core Chess Logic**: Implemented in Python, ensuring a solid and reliable gameplay experience.
- **Interactive Frontend**: Developed using TypeScript to provide a responsive and user-friendly interface.
- **Advanced AI Opponent**: Incorporating AI techniques for challenging and engaging gameplay.
- **Online Multiplayer**: Play against others over the internet for a competitive experience.
- **Chess Engine Enhancements**:
  - Iterative Deepening: For more efficient search.
  - Null Move Pruning: To reduce the number of branches in the game tree.
  - Futility Pruning: To ignore moves unlikely to affect the outcome.
  - Quiescence Search: For handling positions in a state of flux.
  - Transposition Table: To avoid recalculating positions.
  - Bitboard: For efficient chessboard representation and move generation.
  - Alpha-Beta Pruning: To minimize the search time for the best move.
